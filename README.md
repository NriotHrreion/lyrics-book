# Lyrics Book

A web-based lyrics book, including those written by Bobby Chen and Lizhi, powered by fumadocs

**[https://lyrics.nocp.space](https://lyrics.nocp.space)**

## Deploy locally

```
npm run dev
```

Then enter **`http://localhost:3000`** to see the lyrics book
